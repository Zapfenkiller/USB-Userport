Detailed Description

LED definitions for Sparkfun ProMicro.

API to control the 2 LEDs we have on a Sparkfun ProMicro board. Those are taken away from the USB-logic and made available to the user. Only the bootloader takes over when activated. The routines serve as a kind of hardware abstraction layer.
